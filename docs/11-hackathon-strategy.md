# Стратегия хакатона «Games with a Hook» (дедлайн 16.07.2026 03:00 GMT+2)

Исследование: 10 эталонов retention из брифа (живые сабы), доки Devvit, победители 5 прошлых Reddit-хакатонов, инвентаризация репо; идеация по 4 призовым трекам; стресс-тест скептиком и симуляцией жюри. Полные материалы — в журнале workflow за 08.07.

## Ключевые факты исследования

- **Конкуренция реальна, но конечна**: из ~2400 участников до сабмишна доходит 10–18% → ждём 240–420 работ.
- **Тревожный сигнал**: структура призов — клон «Daily Games» (янв 2026). Там $15k взяла игра с UGC-контуром (Soul Thieves — игроки строят башни друг для друга), а чистые дейли-пазлы осели в honorable mentions ($1k). Победители прошлого хакатона (krawlings, BridgeDit) = эталоны следующего.
- **Сквозные паттерны эталонов**: бот-пост как сердцебиение (нумерованный daily); UGC трогает сложность, не косметику; флаир = публичный леджер прогресса (honk: «Rank #2448 | CR 48%»); share-артефакт кодирует ПУТЬ, не ответ (LETTERSET-фраза из твоих слов); закреплённый бот-коммент с правилами; признание по имени; спойлеры лечатся нормой «clue, not answer».
- **Devvit умеет всё нужное**: cron-пост (`scheduler` в devvit.json), Redis + sorted sets (перцентиль = zRank/zCard), `setUserFlair` (флаир-стрики прямо рекомендованы доками), коммент от имени игрока (`runAs:'USER'` — только после ревью-апрува; паттерн: reply к закреплённому бот-комменту), `showShareSheet` с deeplink, спойлер-разметка `>!…!<`. Inline-webview: только tap (без drag), <1 с загрузка. Ревью 1–2 рабочих дня; без него апп ставится только на сабы <200 подписчиков.
- **Симуляция жюри**: база (порт как есть) ≈ 15/40 — зона HM. Со стеком ниже ≈ 31/40 — конкурентно за $15k. Главный рычаг — не фичи, а снятие штрафов.

## Позиционирование

> A daily detective ritual where the community sets the difficulty and the thread compares the paths — the verdict is one, but every detective's trail is their own.

Не «ежедневный логический пазл» (самая населённая ниша, потолок HM), а community-ритуал с пазлом внутри. С этого начинается Devpost-текст и шапка демо-поста.

## TIER 0 — обязательный скелет (без него провал)

1. Webview-клиент из прототипа в logic-thread/ (inline: tap-only, <1 с, 390px).
2. **Полный английский + де-Reddit-рескин** (flair→coat/badge, ModBot→человек, сабреддиты→районы города) — по клиенту И по TS-слою (bank.json, themes.ts, render.ts). Снятие документированного штрафа «on-the-nose theming»: +2 Reddit-y.
3. **Де-слоп**: удалить моки («25 910 детективов», «лучший 01:58»), кнопки-заглушки, `confirm()`. Не «дожать» — удалить.
4. Честные Redis-статы: время→zset→перцентиль/гистограмма, серия, серверный UTC-отсчёт; при N<50 — «You're the 7th detective».
5. Cron-пост «Case #N» + закреплённый бот-коммент: правила в 2 строки, спойлер-норма «clues, not answers; solutions in >!…!<».
6. Онбординг 15 секунд: 3 коуч-марка поверх настоящего дела; в каждом деле — гарантированный первый вывод (ne-улика).
7. Логаут-деградация: ядро играбельно без userId.
8. **Сабмит на ревью к 10.07** + прогон на реальном телефоне в день 1.
9. 8–10 английских дел (флейвор-кит: пулы имён + шаблоны, не проза) и 5–6 живых постов в демо-сабе к 15.07.

## TIER 1 — стек главного приза (~3 дня поверх скелета)

1. **След детектива** (1 д): спойлер-фри строка пути — порядок закрытий, нетронутые улики, время («Case #14 · 3:47 · ③→①→④→② · clue #8 untouched»). Copy + `showShareSheet`; reply к pinned-комменту через `runAs:'USER'` — бонус при апруве, не зависимость. Ответ у всех один — дорога у каждого своя: комменты не дублируются. Reddit-y +2, Hook-y +1; закрывает дыру «нет Wordle-сетки».
2. **Вердикт комьюнити + запечатанное дело** (1–1.5 д, один блок экрана результата): голос решивших «завтра: жёстче/так же/мягче» → cron берёт тир из банка; рядом «конверт» завтрашнего дела (заголовок, тикер «HARD 62%», улика под плашкой, отсчёт). Шапка завтрашнего поста: «The community voted HARD (62%)». Hook-y +3.5: meaningful choice + видимая причинность + предвкушение в первой же сессии судьи. Проходит фильтр решения 28: тир = другая структура улик. HARD = верхний yellow (red-тир 24–25 улик не влезает).
3. **Эпилог + флаир-ранг** (1 д): карточка вчера в сегодняшнем посте («🥇 u/X 2:41 · 214 detectives · voted HARD 62%») + настоящий `setUserFlair` «🔎 Inspector · 🔥 12»; первые 2 повышения — в первые 2–3 дня. Признание по имени (паттерн alignmentchartFills) + публичный леджер (паттерн honk).

## Топ-3 риска

1. **Ревью не апрувнуто к судейству** → сабмит минимальной версии 10.07; все коммент-фичи с фолбэком copy+share sheet.
2. **Английский флейвор — узкое горло** (не параллелится) → флейвор-кит вместо прозы; минимум 6 дел; по одному в день с 10.07.

## Судейская пере-симуляция артефакта (09.07) — новая дыра: ЗАПЕРТЫЙ ХУК

Панель из 5 судей (по одному на критерий) + красная команда, читавших РЕАЛЬНЫЙ `prototype/index.html` (не доки). Итог: **7 / 7 / 7 / 7 из 10** (Delightful UX / Polish / Reddit-y / Hook-y); Phaser 2/10 (форфейт осознанный, не в счёт). Перевод: **уверенно выше медианы, но по всем осям «competitive-but-not-standout» — потолок HM, не $15k.**

**Главная НЕзапланированная дыра (сходится у всех агентов):** весь дифференциатор — вердикт-голос, запечатанное дело, след-трейл — отрисовывается **только после полного решения** (кнопка результата `hidden`, пока не сведены все 12). Дело решается 3–5 мин; судья на холодной 3-мин сессии с телефона до него не доходит → видит «ещё один дейли логик-грид» и кладёт в заезженное ведро дейли-пазлов (исторически = $1k HM, тогда как UGC-петля = $15k). **Это прямое нарушение собственного принципа выше — «каждая фича видима судье за 3 минуты».**

→ **Приоритет №0 поверх TIER 1 (раньше следа детектива):** расшить хук ДО решения. Community-причинность («вчера тред выбрал HARD — ты решаешь завтрашнее») и идея «one verdict, many roads» должны быть видны/ощутимы в первые 20–30 сек: в тексте демо-поста, в стартовом состоянии игры и «подглядыванием» на result-карточку до solve. Без этого фикса остальной стек TIER 1 бьёт мимо судьи, который отваливается на 2:30.

**Вторичные (по убыванию рычага):**
- **Trivia-ведро — риск HIGH.** Не потому что trivia (это дедукция, не эрудиция), а потому что усталый судья снап-категоризирует. Усугубляют: детерминированный ответ (тред сходится, а не расходится) + дифференциатор невидим до solve. Инокуляция = фикс №0 + вести везде общиной-причинностью, не «daily logic puzzle».
- **Моки всё ещё в артефакте** (u/frog_lord, гистограмма на 25 910, посеянные голоса) — TIER-0-удаление ещё НЕ применено к текущему файлу. Проверить на девайсе, что сабмит-дефолт = честный N<50, а не «at scale».
